最想弄懂的5个问题
1.在一个软件开发的过程中，每个人所扮演的角色以及对应的工作的差别

2.在开发过程中，一个团队不同工作需要的人的人数大概是多少

3.在实际开发中，与理论学习上的差距能有多大

4.在一个软件周期中，耗费时间最长，资源最多的哪一个部分
